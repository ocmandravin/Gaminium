What Is GAMINIUM

GAMINIUM is the first cryptocurrency with a guaranteed price floor anchored to global energy markets. The floor is calculated every 2016 blocks from 55+ government data sources across 43 countries, AI-validated by a decentralised oracle network, and defended by an autonomous treasury.

Quantum resistant from genesis block one. Every wallet uses CRYSTALS-Dilithium — the NIST 2024 post-quantum digital signature standard — so your holdings are safe against both classical and quantum computer attacks.

Fixed supply. Community governed. Nobody owns it. 51,200,000 GMN. Hard cap. Forever.

Get Started

How To Mine

RandomX is CPU-friendly and ASIC-resistant. Anyone with a standard computer can mine. No special hardware required.

1
Install Go
# macOS
brew install go

# Other platforms → golang.org/dl
2
Clone and Build
git clone https://github.com/ocmandravin/Gaminium.git
cd Gaminium
go mod tidy && make build
3
Create Wallet
./build/gmn-wallet new

# Save your 24-word mnemonic safely — it cannot be recovered if lost
4
Start Full Node
./build/gaminium start
5
Start Mining — New Terminal
./build/gmn-mine GMN1YOUR_ADDRESS_HERE

Core Innovation

The Price Floor

GAMINIUM's price floor is a cryptographically enforced minimum value, calculated every 2016 blocks from the real cost of energy globally. If the market price drops to the floor, the autonomous treasury begins buying GMN automatically.

No commodities. No gold. No oil. Pure energy.
