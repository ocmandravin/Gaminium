# GAMINIUM Wallet Guide

## Create New Wallet
./build/gmn-wallet new

Generates:
- 24 word mnemonic phrase
- Your GMN1 address

## IMPORTANT — Save Your Mnemonic
Write it down on paper
Store it safely offline
Never share it with anyone
Cannot be recovered if lost

## Validate Address
./build/gmn-wallet validate GMN1YOUR_ADDRESS

## Address Format
Starts with GMN1
Followed by uppercase letters and numbers
Generated using CRYSTALS-Dilithium
Quantum resistant from genesis block one
