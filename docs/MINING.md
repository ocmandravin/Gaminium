# GAMINIUM Mining Guide

## How To Mine

Step 1 — Create wallet:
./build/gmn-wallet new
Copy your GMN1 address

Step 2 — Start node:
./build/gaminium start

Step 3 — Mine (new terminal):
./build/gmn-mine GMN1YOUR_ADDRESS_HERE

## Block Rewards
Era 1 (Year 1-4):   50 GMN per block
Era 2 (Year 5-8):   25 GMN per block
Era 3 (Year 9-12):  12.5 GMN per block
Halving every 420,000 blocks (~4 years)

## Algorithm
RandomX — CPU friendly, ASIC resistant
Anyone with a standard computer can mine
No special hardware needed

## Supply
Total: 51,200,000 GMN (hard cap)
Sub unit: 1 GMN = 100,000,000 Minium
